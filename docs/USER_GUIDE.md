## 1. 快速開始

### 1.1 登入系統

1. 開啟瀏覽器，輸入系統網址（例：`http://localhost:4000`）
2. 在登入頁面輸入您的 **Email** 和**密碼**
3. 點擊「登入」按鈕

**若帳號已啟用雙因素驗證 (2FA)**:
- 輸入 Email 和密碼後，系統會跳轉至 2FA 驗證頁面
- 開啟 Google Authenticator，查看對應帳號的 6 位數驗證碼
- 在驗證頁面輸入驗證碼後，點擊「驗證」完成登入

**登入失敗處理**:
- 密碼連續輸入錯誤 5 次後，帳號將被鎖定，需聯絡管理員解鎖
- 若忘記密碼，請聯絡您的系統管理員重設

### 1.2 儀表板介紹

成功登入後，系統會自動導向**儀表板**頁面。儀表板提供您企業業務狀況的即時概覽。

**儀表板主要區塊**:

| 區塊 | 說明 |
|------|------|
| KPI 卡片 | 顯示關鍵業務指標，包含本月銷售額、待處理訂單數、庫存總量、應收帳款等 |
| 業務趨勢圖 | 近 12 個月的銷售走勢折線圖，幫助您掌握業務成長趨勢 |
| 待辦事項 | 列出需要您處理的事項，例如待審核採購申請、低庫存警示、逾期應收帳款 |
| 快速導航 | 常用功能的快速入口，節省尋找功能的時間 |

**左側導覽列**:
- 點擊各模組名稱可切換至對應功能
- 管理員可見「管理後台」區塊（用戶管理、租戶管理、系統管理）
- 點擊左下角的折疊按鈕可收起/展開導覽列

**右上角工具列**:
- 通知鈴鐺：查看系統通知
- 個人頭像：點擊進入帳號設定、切換語言或登出

## 2. 用戶設定

點擊右上角的個人頭像，選擇「帳號設定」進入設定頁面。

### 2.1 個人資料

在「個人資料」分頁中，您可以修改以下資訊：

- **姓名**: 顯示在系統各處的您的名稱
- **電話**: 聯絡電話（選填）
- **部門**: 所屬部門（由管理員設定，一般用戶無法修改）

修改完成後，點擊「儲存變更」按鈕。

### 2.2 變更密碼

在「安全性」分頁中，您可以更新登入密碼：

1. 輸入**目前密碼**
2. 輸入**新密碼**（需符合密碼複雜度要求）
3. 再次輸入**確認新密碼**
4. 點擊「變更密碼」

**密碼複雜度要求**:
- 最少 8 個字元
- 包含至少一個大寫字母（A-Z）
- 包含至少一個小寫字母（a-z）
- 包含至少一個數字（0-9）
- 包含至少一個特殊字元（例：!@#$%^&*）

### 2.3 語言設定

系統目前支援以下介面語言：

- 繁體中文（預設）
- English

在「外觀」分頁中，點擊語言下拉選單選擇偏好的語言，選擇後介面即時切換，無需重新整理頁面。

## 3. 雙因素驗證設定 (2FA)

雙因素驗證（2FA）為您的帳號增加一層安全防護。啟用後，每次登入除了輸入密碼外，還需要輸入手機 Authenticator App 產生的 6 位數驗證碼，即使密碼洩露，帳號也不會被入侵。

**建議所有管理員帳號強制啟用 2FA。**

### 3.1 準備工作

在設定 2FA 之前，請先在您的智慧型手機安裝以下任一 Authenticator App：

- **Google Authenticator**（推薦）: [iOS App Store](https://apps.apple.com) / [Google Play](https://play.google.com)
- Microsoft Authenticator
- Authy

### 3.2 啟用 2FA — 步驟說明

**步驟一：進入設定頁面**

1. 點擊右上角個人頭像
2. 選擇「帳號設定」
3. 點擊「雙因素驗證」分頁

**步驟二：開始設定**

在「雙因素驗證」分頁，您會看到目前 2FA 狀態為「未啟用」。

1. 點擊「**設定 2FA**」按鈕
2. 系統顯示 QR Code 和手動輸入金鑰頁面

**步驟三：掃描 QR Code**

1. 開啟手機上的 **Google Authenticator**
2. 點擊右下角「+」按鈕
3. 選擇「**掃描 QR Code**」
4. 用相機掃描畫面上的 QR Code
5. Google Authenticator 成功新增帳號，會顯示名稱（例：ERP 全家桶 - your@email.com）和 6 位數動態驗證碼

> 若無法掃描 QR Code，可選擇「手動輸入」，將畫面上的「金鑰」（Secret Key）手動輸入至 Authenticator App。

**步驟四：驗證並啟用**

1. 在手機 Google Authenticator 上查看目前顯示的 6 位數驗證碼
2. 在設定頁面的「驗證碼」欄位輸入這 6 位數字
3. 注意：驗證碼每 30 秒更新一次，請確保在碼更新前完成輸入
4. 點擊「**啟用 2FA**」按鈕
5. 系統顯示「雙因素驗證已成功啟用」確認訊息

**步驟五：完成設定**

2FA 啟用後：
- 頁面顯示 2FA 狀態為「**已啟用**」
- 下次登入時，輸入密碼後系統會要求您輸入 Authenticator 驗證碼

> **重要提醒**: 請妥善保管您的手機。若手機遺失，您將無法登入，需聯絡管理員重設 2FA。

### 3.3 停用 2FA

若您需要停用 2FA（不建議，特別是管理員帳號）：

1. 進入「帳號設定」→「雙因素驗證」分頁
2. 點擊「**停用 2FA**」按鈕
3. 輸入目前 Authenticator App 顯示的驗證碼以確認身份
4. 點擊確認，2FA 停用完成

### 3.4 緊急情況：無法存取 Authenticator

若您遺失手機或無法使用 Authenticator App，請聯絡您的系統管理員（或租戶管理員），請他們執行「重設 2FA」操作。重設後，您可以使用原本的密碼登入，並重新設定 2FA。

## 4. 管理後台

管理後台功能**僅限管理員**（租戶管理員或系統管理員）使用。一般用戶不會在左側導覽列看到此區塊。

### 4.1 用戶管理

**進入方式**: 左側導覽列 → 「管理後台」→「用戶管理」

#### 查看用戶列表

用戶管理頁面顯示您租戶內所有帳號的列表，包含：
- 姓名與 Email
- 角色（系統管理員 / 租戶管理員 / 一般用戶）
- 帳號狀態（啟用 / 停用 / 鎖定）
- 2FA 狀態
- 最後登入時間

您可以使用頁面頂部的搜尋框依姓名或 Email 搜尋用戶，也可以使用下拉篩選器依角色或狀態篩選。

#### 新增用戶

1. 點擊頁面右上角的「**新增用戶**」按鈕
2. 填寫用戶資料：
   - **姓名**（必填）
   - **Email**（必填，用於登入）
   - **初始密碼**（必填，建議通知用戶後立即更改）
   - **角色**（必填）
   - 部門、電話（選填）
3. 若需要指定此用戶可存取的功能，在「權限」區塊勾選對應的功能模組
4. 點擊「**建立用戶**」完成

新用戶建立後，系統狀態預設為「啟用」，需將初始密碼告知新用戶，並建議他們首次登入後立即修改密碼。

#### 修改用戶

1. 在用戶列表中找到要修改的用戶
2. 點擊該行右側的「**編輯**」按鈕（鉛筆圖示）
3. 修改所需的資料
4. 點擊「**儲存變更**」

可修改的資料包含：姓名、角色、部門、電話、密碼（留空表示不變更）、權限設定。

#### 停用用戶

當員工離職或帳號需要暫停使用時：

1. 在用戶列表中找到該用戶
2. 點擊操作選單（三個點的圖示）
3. 選擇「**停用帳號**」
4. 確認操作

停用後該用戶無法登入，但資料保留不刪除。需要時可重新啟用。

#### 啟用用戶

對於停用或被鎖定的帳號：

1. 在用戶列表中找到該用戶（可用「狀態：停用/鎖定」篩選）
2. 點擊操作選單
3. 選擇「**啟用帳號**」
4. 確認操作

#### 重設用戶 2FA

當用戶無法存取 Authenticator App（例如換手機、手機遺失）：

1. 在用戶列表找到該用戶
2. 點擊操作選單
3. 選擇「**重設 2FA**」
4. 確認操作

重設後，該用戶的 2FA 配置被清除，下次登入只需輸入密碼即可（不會再要求 2FA），用戶可重新設定 2FA。

#### 刪除用戶

> **注意**: 刪除操作不可恢復。通常建議使用「停用」而非刪除。

1. 在用戶列表找到要刪除的用戶
2. 點擊操作選單
3. 選擇「**刪除**」
4. 系統顯示二次確認對話框，確認後點擊「確認刪除」

---

### 4.2 租戶管理

> 此功能**僅限系統管理員**使用，租戶管理員看不到此選項。

**進入方式**: 左側導覽列 → 「管理後台」→「租戶管理」

租戶管理頁面顯示平台上所有租戶（客戶企業）的列表。

#### 查看租戶

租戶列表顯示每個租戶的：
- 租戶名稱
- 訂閱方案（基本 / 標準 / 企業）
- 狀態（啟用 / 暫停）
- 用戶數量
- 建立時間

點擊租戶名稱可查看該租戶的詳細資訊。

#### 新增租戶

1. 點擊「**新增租戶**」按鈕
2. 填寫租戶基本資料：租戶名稱、訂閱方案
3. 填寫初始管理員帳號資訊：Email、姓名、初始密碼
4. 填寫聯絡人資訊（選填）
5. 點擊「**建立租戶**」

系統會自動為新租戶建立獨立的資料庫 Schema，此過程可能需要數秒鐘。

#### 暫停租戶

當租戶訂閱到期或需要暫時停止服務時：

1. 找到要暫停的租戶
2. 點擊操作選單 → 選擇「**暫停租戶**」
3. 確認操作

暫停後，該租戶的所有用戶將無法登入，直到重新啟用。

#### 啟用租戶

對暫停中的租戶：

1. 找到該租戶
2. 點擊操作選單 → 選擇「**啟用租戶**」
3. 確認操作

#### 匯出 CSV

1. 點擊頁面右上角的「**匯出 CSV**」按鈕
2. 瀏覽器自動下載 CSV 檔案，檔名格式為 `tenants_export_YYYYMMDD.csv`

匯出的 CSV 包含所有租戶的 ID、名稱、Schema、方案、狀態、用戶數及建立時間。

### 4.3 系統管理

> 此功能**僅限系統管理員**使用。

**進入方式**: 左側導覽列 → 「管理後台」→「系統管理」

#### 資料庫備份

系統管理頁面提供手動下載完整資料庫備份的功能。

**操作步驟**:

1. 進入系統管理頁面
2. 找到「**資料庫備份**」區塊
3. 點擊「**下載資料庫備份**」按鈕
4. 系統在後端執行備份（可能需要數秒至數十秒）
5. 完成後自動下載 `.sql` 備份檔案

**備份檔案說明**:
- 格式：PostgreSQL SQL 腳本（`.sql`）
- 內容：包含平台資料及所有租戶資料
- 檔名格式：`backup_YYYYMMDD_HHmmss.sql`（例：`backup_20260328_113045.sql`）

**安全提醒**:
- 備份檔案包含所有用戶資料，請妥善保管，不要分享給未授權人員
- 建議下載後立即儲存至安全的備份儲存媒體（外接硬碟、加密雲端儲存等）
- 建議每日執行一次備份並保留最近 30 天的備份

## 5. 各模組使用說明

### 5.1 儀表板

儀表板是您登入後的起始頁面，提供企業整體業務狀況的一覽。

**主要功能**:
- 即時顯示關鍵 KPI（今日銷售額、本月採購金額、倉庫庫存量、待處理訂單）
- 業務趨勢折線圖，呈現近 12 個月的銷售走勢
- 待辦清單提醒：待審核申請、低庫存警示、逾期帳款
- 各模組的快速連結入口

---

### 5.2 採購管理

管理企業的採購作業，從供應商維護到訂單建立及付款追蹤。

**主要使用流程**:
1. **維護供應商資料** → 供應商管理 → 新增供應商
2. **建立採購申請** → 採購申請 → 新增申請 → 提交審核
3. **審核通過後轉為採購訂單** → 採購訂單管理
4. **收貨確認** → 訂單詳情 → 確認到貨數量
5. **登錄供應商發票** → 應付帳款 → 新增應付款

---

### 5.3 銷售管理

管理完整的銷售流程，從報價到收款。

**主要使用流程**:
1. **維護客戶資料** → 客戶管理 → 新增客戶
2. **建立報價單** → 報價管理 → 新增報價 → 發送給客戶
3. **客戶確認後轉為銷售訂單** → 銷售訂單
4. **安排出貨** → 訂單詳情 → 確認出貨
5. **開立發票、收款** → 應收帳款

---

### 5.4 庫存管理

即時掌握貨品庫存狀況，管理各倉庫的進出貨作業。

**主要功能**:
- **庫存查詢**: 依商品或倉庫查詢目前庫存量、可用量
- **庫存異動**: 手動入庫、出庫、倉庫間調撥
- **庫存盤點**: 建立盤點任務，記錄實際盤點數量，自動計算差異
- **低庫存警示**: 庫存低於安全庫存量時，儀表板顯示警示

---

### 5.5 生產管理

管理製造流程，從 BOM 設計到工單執行與完工入庫。

**主要使用流程**:
1. **設定 BOM（物料清單）** → BOM 管理 → 建立多層 BOM
2. **建立工單** → 生產工單 → 依銷售訂單或計劃建立
3. **領料作業** → 工單詳情 → 執行物料領料
4. **工序回報** → 各工序完成後回報進度
5. **完工入庫** → 工單完工 → 成品自動入庫

---

### 5.6 財務管理

處理企業的帳務與財務報表。

**主要功能**:
- **傳票管理**: 手動建立會計傳票，或由其他模組自動產生
- **帳款管理**: 追蹤應收帳款與應付帳款，進行收付款登錄
- **財務報表**: 一鍵生成損益表、資產負債表、現金流量表
- **帳齡分析**: 分析逾期未收/未付的帳款

---

### 5.7 人力資源

管理員工資料、考勤及薪資。

**主要功能**:
- **員工檔案**: 維護員工基本資料、職位、薪資結構
- **組織架構**: 維護部門樹狀圖與匯報關係
- **出勤管理**: 查看員工打卡記錄、請假申請審核
- **薪資計算**: 每月執行薪資計算，生成薪資條
- **招募管理**: 發布職缺、追蹤應徵進度

---

### 5.8 客戶關係管理 (CRM)

管理客戶互動與銷售機會。

**主要功能**:
- **聯絡人管理**: 維護客戶聯絡人完整資料與互動歷史
- **銷售漏斗**: 追蹤各階段的商機進度（潛在 → 提案 → 協商 → 成交）
- **活動記錄**: 記錄電話、Email、拜訪等客戶互動
- **客服工單**: 處理客戶投訴與服務請求

---

### 5.9 品質管理

建立與執行品質管理體系。

**主要功能**:
- **檢驗計劃**: 設定進料、製程、出貨的檢驗標準
- **品質檢驗**: 執行檢驗並記錄檢驗結果（合格/不合格）
- **不合格品處理**: 記錄不良品處置方式（退貨/讓步接收/報廢）
- **品質報告**: 查看各供應商與產品的品質趨勢分析

---

### 5.10 商業智能 (BI)

數據分析與視覺化報表。

**主要功能**:
- **預建報表**: 銷售分析、採購分析、庫存周轉、財務摘要等預建報表
- **自定義報表**: 使用報表設計器建立專屬分析報表
- **KPI 儀表板**: 設定個人化的 KPI 監控儀表板
- **資料匯出**: 將報表匯出為 Excel、CSV 或 PDF 格式

---

### 5.11 流程管理 (BPM)

設計與管理業務審核流程。

**主要功能**:
- **流程設計**: 使用圖形化設計器建立審核流程
- **流程執行**: 提交流程申請，系統自動路由至審核人
- **待辦任務**: 查看待您審核或填寫的任務
- **流程追蹤**: 查看流程目前進行至哪個步驟

### 5.12 POS 銷售

實體門市收銀操作。

**操作說明**:

**開班**:
1. 進入 POS 模組
2. 選擇收銀機與班次
3. 輸入開班現金金額
4. 點擊「開始班次」

**收銀操作**:
1. 搜尋商品（輸入品名或掃描條碼）
2. 點擊商品加入購物車
3. 調整數量（若需要）
4. 選擇付款方式（現金/信用卡/電子錢包）
5. 輸入收款金額（現金交易），系統計算找零
6. 點擊「完成交易」
7. 列印/傳送電子收據（選填）

**退貨處理**:
1. 查詢原始交易記錄
2. 選擇要退貨的商品
3. 確認退貨原因
4. 完成退款

**結班**:
1. 點擊「結束班次」
2. 核對現金金額
3. 確認班次銷售彙總
4. 點擊「確認結班」
